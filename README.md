# Hangman

A CLI [Hangman](https://en.wikipedia.org/wiki/Hangman_(game)) implementation in ruby.

## About

This project is part of the Ruby Programming lessons of [The Odin Project](https://www.theodinproject.com/lessons/ruby-hangman) curriculum, with the sole purpose of practicing File serializtion in ruby.

## Features

- **Save** You can save your game progress at the start of any turn.
- **Load** Upon starting the game, you can load last saved game.
